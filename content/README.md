# ヨシレンタカー コンテンツ共有

現在のWordPressサイトから移行するコンテンツを整理するディレクトリです。

## ディレクトリ構成

- **images/** - 画像ファイル（ロゴ、サービス写真、バナー等）
- **texts/** - テキスト内容（ページテキスト、説明文等）

## 使用方法

1. WordPressサイトの画像をダウンロードして、imagesフォルダに保存
2. テキスト内容をファイルまたはMarkdownで整理して、textsフォルダに保存
3. 内容が共有されたら、このファイルを更新して進捗を記録

- [ ] マイクロバスレンタルサービスの説明

- [ ] サービス写真
- [ ] その他バナー・アイコン画像
